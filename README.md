# 🚀 TaskPilot — Smart Productivity To-Do App

TaskPilot is a modern, feature-rich, and fully responsive To-Do List web application designed to help users efficiently manage daily activities, organize schedules, track productivity, and improve workflow management.

Built using **HTML, CSS, and Vanilla JavaScript**, the application focuses on delivering a clean and interactive user experience with modern UI/UX principles, responsive layouts, real-time filtering, progress analytics, dark/light theme support, and local data persistence.

# 🌟 Key Features

## ✅ Smart Task Management

The application allows users to efficiently manage tasks by:

- Adding new tasks
- Assigning task priorities
- Setting due dates
- Marking tasks as completed
- Managing pending tasks
- Deleting unnecessary tasks

TaskPilot provides a smooth and intuitive productivity workflow for daily task management.

# 🎯 Priority-Based Task Organization

Tasks are visually categorized using color-coded priority indicators for better task planning and focus.

| Priority Level | Indicator |

| High Priority  | 🔴 Red Border |
| Medium Priority | 🟡 Yellow Border |
| Low Priority | 🟢 Green Border |

This helps users quickly identify important and urgent tasks.
